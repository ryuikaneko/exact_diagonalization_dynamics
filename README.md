# calculate dynamics for quantum many body systems

* 1D transverse field Ising model
  * Usage
    ```console
    foo@bar:~$ python quench_dynamics_1d_FM_TFIsing.py -N [system size]
    ```
    The result for the system size N=16 is shown as an example.
